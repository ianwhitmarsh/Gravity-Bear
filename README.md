# Choose GRAVITY BEAR Landing Page

Static landing page for GRAVITY BEAR adult membership and 2-week trial ads in Tulsa. The page uses hosted GRAVITY BEAR images so it can deploy from a tiny static repo.

Open `index.html` in a browser to view the page, or run:

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```
